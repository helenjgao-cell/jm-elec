# 浙江精铭电子科技有限公司 - 企业官网

## 网站说明

这是一个为浙江精铭电子科技有限公司制作的企业官方网站，采用纯HTML/CSS/JS静态页面构建，无需后端服务器即可部署。

## 网站结构

```
jingming-website/
├── index.html          # 首页
├── about.html          # 关于我们
├── products.html       # 产品中心
├── contact.html        # 联系我们
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互脚本
└── images/             # 图片文件夹（待添加）
```

## 网站功能

- **首页**：公司介绍、产品展示、联系方式预览
- **关于我们**：公司简介、企业价值观、发展历程
- **产品中心**：12款主营产品展示，支持分类筛选
- **联系我们**：在线留言表单、联系方式、公司地址

## 部署方式

### 方式一：静态托管（推荐）
将网站文件上传到以下任一平台：
- GitHub Pages（免费）
- Netlify（免费）
- Vercel（免费）
- 阿里云/腾讯云对象存储

### 方式二：自有服务器
1. 将文件上传到服务器网站目录
2. 配置Web服务器（Nginx/Apache）
3. 绑定域名

### 方式三：本地预览
直接在浏览器中打开 `index.html` 文件即可预览效果。
